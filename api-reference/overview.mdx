## Overview

Glo Exchange provides REST APIs for programmatic trading, account management, and market data access.

## Base URLs

| Environment | URL |
|------------|-----|
| Production | `https://api.glo.exchange` |

## Authentication

Most API endpoints require authentication via JWT bearer token. See the [Authentication](/api-reference/authentication) page for details.

## Rate Limits

| Endpoint Type | Rate Limit |
|--------------|------------|
| Authenticated | 100 requests / 15 min |
| Public data | 300 requests / 15 min |
