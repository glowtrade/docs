## Public Endpoints

Market data endpoints are publicly accessible without authentication.

### Get All Markets

```
GET /api/config/markets
```

Returns all available perpetual futures markets with current prices, 24h volume, and funding rates.

### Get Asset Price

Prices are sourced in real-time from Hyperliquid's orderbook. Use the markets endpoint to get current mark prices for all assets.

### News

```
GET /api/cryptonews/headlines/:ticker
GET /api/stocknews/headlines/:ticker
GET /api/forexnews/headlines
```

Fetch the latest news for crypto, stocks, and forex/commodity assets.

<Info>
  For real-time price streaming, use Hyperliquid's WebSocket API directly at `wss://api.hyperliquid.xyz/ws`.
</Info>
