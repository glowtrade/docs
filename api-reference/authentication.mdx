## Authentication Method

The Glo Exchange API uses **JWT Bearer tokens** for authentication. Include the token in the `Authorization` header of your requests.

```bash
curl -X GET https://api.glo.exchange/api/user/profile \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

## Obtaining a Token

Authentication is handled via Privy. When a user signs in through the web or mobile app, a JWT token is issued that can be used for API access.

<Warning>
  Keep your JWT token secure. Never share it or expose it in client-side code.
</Warning>
